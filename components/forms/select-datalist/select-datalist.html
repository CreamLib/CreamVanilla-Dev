Regular Select

title
The Label of the Input
name
The value of the name attribute
id
The id of the Input
for
Need to be the same of the id
                            
                                <!-- REGULAR SELECT -->
                                <p class="ui-select">
                                    <label for="choice">Choose an answer</label>
                                    <select id="choice" name="choice">
                                        <option value="An answer">An answer</option>
                                        <option value="Another answer">Another answer</option>
                                        <option value="A 3rd answer">A 3rd answer</option>
                                        <option value="The right answer">The right answer</option>
                                    </select>
                                </p>
                            
                        
                            .toto {
                                color: #FFF;
                            }

Datalist

for
Reference to the id of the Input
name
The value of the name attribute
id
The id of the Input
list
Link between input and datalist
value
Value of the option's datalist
                            
                                <!-- REGULAR DATALIST -->
                                <p class="ui-select-datalist">
                                    <label for="choose-me">Choose an answer</label>
                                    <input type="text" id="choose-me" name="choose-me" list="mylist1" />
                                    <datalist id="mylist1">
                                        <option value="An Answer"> An Answer </option>
                                        <option value="Another answer"> Another answer </option>
                                        <option value="A 3rd answer"> A 3rd answer </option>
                                        <option value="The right answer"> The right answer </option>
                                    </datalist>
                                </p>
                            
                        
                            .toto {
                                color: #FFF;
                            }

Combined Datalists

list
Refer to the same datalist
                            
                                <!-- COMBINED DATALIST -->
                                <div class="ui-select-datalists">
                                    <p class="ui-select-datalist">
                                        <label for="choose-me1">Choose me please</label>
                                        <input type="text" id="choose-me1" name="choose-me1" list="mylist2" />
                                    </p>
                                    <p class="ui-select-datalist">
                                        <label for="choose-me2">Choose me too</label>
