Tooltip Slider

0

Just add the class bubble when calling the ui-slider component to display the output element as a tooltip.

                            
                                <!-- SLIDER -->
                                <p class="ui-slider">
                                    <label for="slider1">Choose a value</label>
                                    <!-- Input type RANGE -->
                                    <input type="range" id="slider1" min="0" max="50" step="1" value="25" />
                                    <!-- Output for display change value -->
                                    <span for="slider1" class="output" id="output">0</span>
                                </p>
                            
                        
                            .toto {
                                color: #FFF;
                            }
