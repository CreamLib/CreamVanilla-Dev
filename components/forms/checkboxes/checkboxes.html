Checkboxes

type
The type of the Input
id
The most important attribute : Link between Label and Input
name
The name of input
for
Must be the same of the id

Custom checkbox (add the custom class)

Unstressed checkbox (add the unstressed class)

On-off Button (add the on-off class)

            
            <!-- Normal checkbox -->
            <p class="ui-checkboxes">
              <input type="checkbox" id="checkbox1" name="checkboxName1">
              <label for="checkbox1">
            </p>
            

            <!-- Custom checkbox -->
            <p class="ui-checkboxes custom">
              <input type="checkbox" id="checkbox2" name="checkboxName2">
              <label for="checkbox2">
            </p>
            

            <!-- Unstressed checkbox -->
            <p class="ui-checkboxes unstressed">
              <input type="checkbox" id="checkbox3" name="checkboxName3">
              <label for="checkbox3">
            </p>
            

            <!-- On-off checkbox -->
            <p class="ui-checkboxes on-off">
              <input type="checkbox" id="checkbox4" name="checkboxName4">
              <label for="checkbox4">
            </p>
            
          
              .toto {
                color: #FFF;
              }
